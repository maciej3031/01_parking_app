SIMPLE PARKING APPLICATION

1. Application is developed in Python. Database is created in SQLite. Application language = polish.

2. Application allows to:
- register the car park entrance with date and time of entrance using vehicle registration number
- register the car park exit with calculation of costs
- show the list of parked cars and cars with subscription that are not parked
- change stake, basic parking time, subscription stake, subscription time
- buy or renew subscription with calculation of costs
